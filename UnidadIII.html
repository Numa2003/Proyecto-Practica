Arreglos(Matrices)

Conceptualización

Son un tipo especial de variables que pueden temporalmente uno o varios datos.Una variable.
de arreglo o vector puede contener al mismo tiempo,diferentes valores bajo un mismo nombre.

Matrices

Una Matriz es un tipo de variable que puede almacenar varios valor a la vez.En las matrices de
una dimensión (vectores) cada elemento se identifica por un índice que se escribe entre corchetes
($matriz($indice)).

Podemos imaginarnos a un arreglo como una secuencia contigua de celdas (espacios en memoria) o
casillas,en cada una de las cuales se puede guardar un elemento de la colección.

Un array puede ser creado con el constructor del lenguaje array().Este toma cualquier número de parejas,
clave=>valor como argumentos.

La clave puede ser un integer o un string y el valor puede ser de cualquier tipo.
En la declaración de los Arrays hay que tomar en cuenta las siguientes características:
*En string que contenga un integer válido será convertido al tipo integer. Por ejemplo: Si tenemos la
clave "3",este será almacenada como 3;pero si tenemos la clave "03" no será convertido,ya que no
es un número integer decimal válido.

*Un float también será convertido a integer,lo que significa que la parte fraccionaria se eliminará. Por
ejemplo:Si la clave es 7.2,este será almacenado como 7.

*Cuando la clave es un booleano,estos son convertidos a integers,es decir,la clave true será almacenada
como 1 y la clave false como 0.

Los arrays y los objects no pueden utilizarse como claves.Si se hace,dará lugar a una advertencia:
Illegal offset type.

En caso que varios elementos de un array utilicen la misma clave,solamente será tomada la última.

Como todas las claves en el ejemplo anterior se convierten en 1,los valores serán sobreescritos en cada
nuevo elemento,por lo que el último valor asignado "Alexandra Fares" es el único que queda.

Algo importante que mencionar,es que los arrays en PHP pueden contener claves integer y string al
mismo tiempo ya que no distingue entre arrays indexados y asociativos.

Como hemos observado el índice puede ser de tipo de cadena o númerico,también podemos decir que
cuando se crean dos índices idénticos,el último sobrescribe al primero.

Cuando se omite el índice,se genera un índice numérico automáticamente,empezando por 0.Si el índice
es númeerico,el siguiente índice será el número del índice mayor +1.

Range

Permite crear un array que tiene un rango de elementos.
Sintaxis:
range(inicio,fin,step)

Donde:
*Inicio:Representa el primer valor de la secuencia.
*Fin:La secuencia finalizada al encontrar el valor fin.
*Step:Si se proporciona un valor a step,este será usado como el incremeto entre los elementos de la
secuencia.Si no se especifica,el valor predeterminado de step será 1.
