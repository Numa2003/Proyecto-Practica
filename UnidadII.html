PHPMyAdmin

Es un servidor web HTPP de código abierto para la creación de páginas y servicios web, a su vez,
es un servidor multiplataforma,gratuito y robusto.

PHPMyAdmin permite realizar operaciones básicas en bases de datos MySQL a través de páginas web,utilizando Internet.Está disponible bajo la licencia GPL(General Public License) y en más de 50 idiomas.

PHPMyAdmin permite realizar operaciones básicas en bases de datos MySQL como pueden ser:
*Crear y eliminar bases de datos
*Crear,eliminar y alterar tablas
*Borrar,editar y añadir campos
*Ejecutar sentencias SQL
*Administrar claves en campos,administrar privilegios y exportar datos en varios formatos

MySQL

Es un sistema de administración de bases de datos relacionales.En consecuencia,una base de datos relacional almacena datosen tablas separadas en lugar de ponersetodos los datos en un sólo lugar,esto agrega velocidad y flexibilidad.Las tablas son enlazadas al definir relaciones que hacen posible combinar datos de varias tablas cuando se necesitan consultar datos.

MySQL es Open Source,en otras palabras,cualquier persona puede usar y modificar MySQL sin ningún
costo.MySQL usa la licencia GLP(Licencia Pública General GNU) y no se puede hacer con el software para diferentes situaciones.MySQL también cuenta con Licencia Comercial para todas aquellas personas que la licencia gratuita no cumple con sus requisitos.

PHP

Es un lenguaje de programación de código abierto y del lado del servidor,originalmente diseñado para el desarrollo web de de contenido dinámico.Algo importante,es que PHP se procesa en servidores.

Por Por regla general,este tipo de lenguaje es utilizado para crear contenido diná y poder interectuar con el usuario.

Puede haber código PHP embedido(incluido o incrustado) en código HTML

Servidor Web

Es un programa que procesa una aplicación del lado del servidor,realizando conexiones bidireccionales y/o
unidireccionales con el cliente y generando o cediendo una respuesta en cualquier lenguaje o aplicación del lado del cliente,sin embargo,estos servidores utilizan el Protocolo HTTP.

En otras palbras podemos decir,que es un programa diseñado para transferir datos de hipertexto,es
decir,páginas web con todos sus elementos(textos,imágenes,banners,etc.).
