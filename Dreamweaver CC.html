Dreamweaver CC

DreamWeaverCC

Es una aplicación que está diseñada para la construcció,diseño y edición de sitios web.

Dreamweaver agrupa todos los archivos creados (Html,Php,Css,etc.) en un proyecto denominado Sitio Web. Al ingresar a Dreamweaver se visualizará la siguiente pantalla:

Crear un Nuevo Archivo

Para crear un nuevo archivo HTML en DreamWeaver,realizar las siguientes instrucciones:
1.En el área Crear nuevo de la pantalla de inicio de DreamWeaver,dar click en HTML o dar click en
Archivo y acontinuación en Nuevo...

Dar click en crear

2.Elegir el tipo de documento HTML que desea crear y dar click en el Botón Crear.

3.Se mostrará la ventana dónde se puede diseñar la página Web ya sea directamente o vía código.

Guardar un Archivo

Para guardar un archivo desde Dreamweaver,realizar las siguientes instrucciones:
1.Dar click en el Menú Archivo y a continuació en Guardar

2.Seleccionar la ubicación,escribir el nombre del archivo y dar click en el Botón Guardar

Tambín se puede crear un archivo en DreamWeaver,realizando las siguientes instrucciones:
1.En el panel Archivos,seleccionar la unidad o sitio dónde se almacenará el archivo.

2.Dar click derecho del mouse sobre la carpeta que contendrá el archivo

3.En el menú contextual que se muestra dar click en Nuevo Archivo

4.Escribir el nombre del archivo y presionar Enter.

Escribir Código en un Archivo

Para poder escribir un documento HTML realizar las siguientes instrucciones:

1.Abrir o crear el archivo sobre el cual se desea trabajar.

2.Dar click en el Botón Dividir para que se muestre la ventana de diseño y la ventana de código.

3.Mientras se escribe el código ( < img src=imagenes/encabezado1.png " /> podemos observar que
DreamWeaver muestra sugerencias.

Al escribir el signo de apertura de etiquetas automáticamente se muestra una lista de etiquetas.

4.Al terminar de escribir el código podremos observar inmediatamente el resultado.

5.Si damos click en la línea de código que acabamos de crear podemos observar que se selecciona la
imagen y que podemos cambiarla desde la barra de propiedades.Si la barra de propiedades no se
muestra dar click en el Menú ventana y a continuación en Propiedades.

6.Guardar los cambios y presionar la tecla F12 para poder visualizar el resultado en el navegador.
