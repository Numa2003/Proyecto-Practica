Funciones

Una función es un conjunto de instrucciones que realizan una tarea determinada,las instrucciones están
agrupadas en un mismo bloque que puede ser llamado desde cualquier parte del código;de esta manera,
las funciones nos ayudan a que no tengamos que repetir cierto código cada vez que deseamos realizar la
misma tarea.

Existen dos tipos de funciones:
*Funciones Predefinidas:Son funciones que ya vienen integradas en el lenguaje de programación,en
este caso en PHP.Como ya existen solamente necesitamos llamarlas como Count.
*Funciones Creadas por el Usuario:Son funciones que no existen pero que el usuario (programador)
puede crear con el objetivo de realizar cálculos complejos,actividades que no están definidas en el
lenguaje,etc.

realice la suma de dos números,necesitará que le pasemos argumentos parámetros esos dos números.

El argumento puede ser cualquier variable y se utilizará dentro de la función para operar con ella y
obtener el resultado.No todas las funciones necesitarán datos como argumentos.

{Instrucciones}:Grupo de cóidgo que permitirá realizar todas las actividades,necesarias para poder
obtener un resultado.
*return:Se utiliza para indicar el valor que devolverá la función.
