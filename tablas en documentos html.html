Tablas en Documentos HTML

Conceptualización

Las tablas se utilizan para agrupar información en columnas y filas.En versiones anteriores de HTML,las tablas se utilizaban para la maquetación de los sitios,con la aparición de los CSS esta técnica ya no se utiliza.

Cada tabla está compuesta por filas y columnas,el conjunto de la tabla se delimita con las directicas < table> < /table> ,la filas se crean con la directiva < tr> y las celdas con la etiqueta < td>.

Las directivas < tr> y < td> trabajan siempre juntas aunque haya que crear solamente una celda.

Aplicar Bordes a las Tablas

Para aplicar un borde a una tabla se utiliza el atributo < border> de la directiva < table> entre más grande es
el número del valor númerico del atributo más grueso es el borde.

HTMl5 solamente permite el atributo border dentro de la directiva < table>.

Para controlar cada aspecto de la tabla,HTML ofrece etiquetas que permiten controlarlo.

Una tabla está formada por:
*Filas se crean con la etiqueta < tr>
*Grupo de columnas se manipulan con la etiqueta ( < colgroup>).
*Encabezamiento de las celdas se crea con la etiqueta < th>.
*Cuerpo de la celda se crea con la etiqueta < th> o < td>.
*Titulo de la tabla se crea con la etiqueta < caption>.
*Encabezamiento de la tabla se agrupa con la etiqueta < thead>.
*Cuerpo de la tabla se agrupa con la etiqueta < tbody>.
*Pie de la tabla se agrupa con la etiqueta < tfoot>.
